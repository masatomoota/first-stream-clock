# StreamClock

黒背景・緑文字の JST デジタルクロック（配信オーバーレイ向け）。
Rust 製ネイティブアプリ（eframe/egui）で、長時間の連続稼働を想定しています。
Windows / macOS 両対応（クロスプラットフォームのクレートのみ使用）。

## 表示

縦4段（上から）:

1. 日付 `YYYY/MM/DD`（小さめ）
2. 現在時刻 `HH:MM:SS`（大きく）
3. ストップウォッチ `HH:MM:SS`（大きく・状態で色が変わる）
4. ステータス行（時刻ソースの同期状態を小さく表示）

ウィンドウのサイズに連動して文字も自動で拡大・縮小します（右下グリップでリサイズ）。
時刻はすべて JST（UTC+9 固定。PCのタイムゾーン設定に依存しません）。

## 時刻ソース

右クリックメニューまたは設定画面で切替できます。

| ソース | 説明 |
|--------|------|
| System | PCのシステム時計（UTC→JST換算） |
| NTP | SNTPで定期同期（既定: `ntp.nict.jp`、64秒間隔）。オフセットを加算して表示 |
| PTP | IEEE 1588 (PTPv2) UDPマルチキャスト受信（AES67 / SMPTE ST 2110系）。Sync/Follow_Up からオフセット算出、Announce の currentUtcOffset で TAI→UTC 換算。受信専用（経路遅延を含むがLANではサブms） |
| MTC | MIDIタイムコード受信（クォーターフレーム + フルフレームSysEx対応、24/25/29.97/30fps自動判別） |
| LTC | SMPTE LTC をオーディオ入力から復号（バイフェーズマーク自前デコーダ、極性・レベル非依存） |

MTC/LTC は信号が2秒途切れるとフリーホイールを止めて最終値を保持し、ステータス行に **NO SIGNAL** を赤表示します。日付行は常にシステム/NTP系のJST日付です。

## 操作

| 操作 | 動作 |
|------|------|
| 文字エリア（ストップウォッチ以外）をドラッグ | ウィンドウ移動 |
| 右下グリップをドラッグ | サイズ変更（文字も連動拡大） |
| マウスホイール / ↑↓キー | 背景（黒）の不透明度を調整 |
| **ストップウォッチをダブルクリック** | **ストップ → リセット → スタート** の順に切替 |
| 右クリック | メニュー（時刻ソース / 最前面切替 / 設定 / 終了） |
| Esc | 終了 |

ストップウォッチの色: 暗い緑=待機 / 明るい緑=計測中 / 橙=停止中。
背景の黒だけ透過し、緑の文字は不透明のまま残ります（0%で文字だけが浮かびます）。

設定（時刻ソース、NTPサーバー、PTPドメイン、MIDIポート、オーディオ入力デバイス、不透明度、最前面、ウィンドウ位置・サイズ）は自動保存され、次回起動時に復元されます。

## ビルド

[Rust](https://rustup.rs/) を導入のうえ:

```sh
cargo build --release
```

- Windows: `target\release\stream-clock.exe`（MSVCツールチェーン。リリースビルドはコンソール非表示）
- macOS: `target/release/stream-clock`（MIDI=CoreMIDI、音声=CoreAudioが自動で使われます）

> Windows 11 の Smart App Control が有効だと、無署名バイナリ（cargo のビルドスクリプト含む）の実行がブロックされます（os error 4551）。開発機ではオフにしてください。

## テスト

```sh
cargo test
```

LTCデコーダはテスト内エンコーダとのラウンドトリップ（30fps/48kHz、25fps/44.1kHz・低レベル・極性反転、偽同期耐性）、MTCはクォーターフレーム組み立て、PTPはパケット解析、NTPはタイムスタンプ換算を検証しています。

## 構成

```
src/
  main.rs  — GUI (eframe/egui)・設定永続化・ストップウォッチ
  tc.rs    — SMPTEタイムコード型
  ntp.rs   — SNTPクライアント（自前実装・依存なし）
  ptp.rs   — PTPv2リスナー（自前実装・依存なし）
  mtc.rs   — MTC受信 (midir)
  ltc.rs   — LTC復号 (cpal + 自前バイフェーズマークデコーダ)
legacy/    — 旧PowerShell+WPF版（v0.1）
```
