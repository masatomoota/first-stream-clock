# StreamClock

黒背景・緑文字の JST デジタルクロック（配信オーバーレイ向け）。
Windows 標準の PowerShell + WPF で動作し、**インストール・ビルド不要**です。

## 表示

縦3段（上から）:

1. 日付 `YYYY/MM/DD`（小さめ）
2. 現在時刻 `HH:MM:SS`（大きく）
3. ストップウォッチ `HH:MM:SS`（大きく）

すべて JST（日本標準時, UTC+9）。PCのタイムゾーン設定に関わらず常に JST を表示します。
ウィンドウのサイズに連動して文字も自動で拡大・縮小します（角の三角グリップでリサイズ）。

## 起動方法

- **`StreamClock.vbs` をダブルクリック**（推奨。コンソールが一切出ません）
- または `StreamClock.bat` をダブルクリック

> 初回に「WindowsによってPCが保護されました」等が出た場合は「詳細情報」→「実行」で起動できます。

## 操作

| 操作 | 動作 |
|------|------|
| 日付/時刻エリアをドラッグ | ウィンドウ移動 |
| 角の三角グリップをドラッグ | サイズ変更（文字も連動して拡大） |
| マウスホイール | 背景（黒）の不透明度を調整 |
| ↑ / ↓ キー | 背景（黒）の不透明度を調整 |
| **ストップウォッチをダブルクリック** | **ストップ → リセット → スタート** の順に切替 |
| 右クリック | メニュー（最前面切替 / 不透明度プリセット / 終了） |
| Esc | 終了 |

### ストップウォッチの状態と色

- 暗い緑：待機（00:00:00）
- 明るい緑：計測中
- 橙：停止中

待機中にダブルクリック → スタート、計測中にダブルクリック → ストップ、停止中にダブルクリック → リセット（00:00:00）。

## 背景の透過について

背景の黒だけ不透明度を下げられます（緑の文字はくっきり不透明のまま）。
0% にすると黒が完全に消え、文字だけが浮かんで下のウィンドウが見えます。
不透明度を下げてもウィンドウのドラッグ・操作は可能です。

## カスタマイズ

`StreamClock.ps1` を編集:

- 文字色：`Foreground="#00FF66"`（緑）を変更
- フォント：`FontFamily="Consolas"` を変更（7セグ風にしたい場合は [DSEG](https://github.com/keshikan/DSEG) フォントを入れて `DSEG7 Classic` などを指定）
- 文字サイズ比：`FontSize`（日付26 / 時刻96 / ストップウォッチ74）の比率を変更
- 既定の透過率：末尾の `Set-BgAlpha 255` を `0`〜`255` で変更
