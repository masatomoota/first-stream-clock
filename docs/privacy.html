StreamClock — プライバシーポリシー

最終更新: 2026年7月10日

収集する情報

StreamClock は、いかなる個人情報も収集・保存・送信しません。 利用状況の解析、広告、クラッシュレポートの送信も一切行いません。第三者への情報提供もありません。

ネットワークの利用

本アプリがネットワークへ接続するのは、利用者が NTP 時刻同期を有効にした場合のみ です。 このとき、利用者自身が設定画面で指定した時刻サーバー（既定値: ntp.nict.jp）に対して、 標準的な NTP/SNTP の時刻問い合わせパケットを送信します。この通信に個人を識別する情報は含まれません。 NTP を無効にしている場合、本アプリは一切ネットワーク通信を行いません。

デバイスへのアクセス

  • オーディオ出力: 利用者が LTC 出力を有効にした場合のみ、指定されたオーディオ出力デバイスへ SMPTE リニアタイムコードの音声信号を再生します。
  • MIDI 出力: 利用者が MTC 出力を有効にした場合のみ、指定された MIDI 出力ポートへ MIDI タイムコードを送信します。
  • マイク・カメラ・位置情報・連絡先には一切アクセスしません。 本アプリは音声を録音しません。

設定の保存

表示色・時刻ソース・選択したデバイス名などの設定は、利用者の Mac 内のアプリ専用領域にのみ保存されます。 外部へ送信されることはありません。

お問い合わせ

本ポリシーに関するご質問は info@firstcall.plus までご連絡ください。

StreamClock — Privacy Policy

Last updated: 10 July 2026

Information we collect

StreamClock collects, stores and transmits no personal information whatsoever. There is no analytics, no advertising, and no crash reporting. Nothing is shared with third parties.

Network use

The app contacts the network only when you enable NTP time synchronisation. In that case it sends standard NTP/SNTP time-query packets to the time server you specify in Settings (default: ntp.nict.jp). These packets contain no personally identifying information. With NTP disabled, the app performs no network communication at all.

Device access

  • Audio output: only when you enable LTC output, the app plays a SMPTE Linear Timecode audio signal to the output device you select.
  • MIDI output: only when you enable MTC output, the app sends MIDI Time Code to the MIDI output port you select.
  • The app never accesses the microphone, camera, location or contacts. It does not record audio.

Settings storage

Preferences such as colour, time source and the names of the devices you selected are stored only in the app's private container on your Mac. They are never transmitted.
